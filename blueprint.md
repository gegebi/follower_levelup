# 샤갈 강화하기

## 개요

'샤갈 강화하기'는 사용자가 '샤갈'이라는 아이템을 강화하는 웹 기반 게임입니다. 사용자는 '샬'이라는 게임 내 재화를 사용하여 강화를 시도합니다. 강화 레벨이 올라갈수록 강화 비용이 증가하고 성공 확률은 감소하여 사용자에게 도전적인 경험을 제공합니다.

## 스타일, 디자인 및 기능

### 초기 버전

*   **게임 상태 표시:**
    *   현재 보유한 '샬' 금액
    *   '샤갈'의 현재 강화 레벨
    *   다음 강화에 필요한 비용
    *   다음 강화의 성공 확률
*   **상호작용:**
    *   '강화' 버튼을 클릭하여 강화 시도
*   **게임 로직:**
    *   초기 '샬': 1000
    *   초기 강화 비용: 100 '샬'
    *   강화 성공 시:
        *   강화 레벨 1 증가
        *   '샬'이 강화 비용만큼 차감
        *   강화 비용 증가
        *   강화 성공 확률 감소
    *   강화 실패 시:
        *   '샬'이 강화 비용만큼 차감
*   **디자인:**
    *   깨끗하고 직관적인 사용자 인터페이스
    *   모바일 반응형 디자인
    *   현대적인 색상 팔레트와 글꼴
*   **레벨별 색상 변경:**
    *   레벨(티어)에 따라 아이템의 색상이 변경되어 시각적으로 구분이 가능하도록 개선되었습니다. 각 티어는 더욱 생생하고 뚜렷한 색상으로 표현됩니다.
